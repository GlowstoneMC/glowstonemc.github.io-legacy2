Simplicity breeds performance. Glowstone++ is built with both of these in mind, making it suited for situations where vanilla game features aren't needed.

About Glowstone++

Glowstone++ is a lightweight, from scratch, open source Minecraft server written in Java that supports plugins written for the Bukkit and Sponge APIs. The main goals of the project are to provide a lightweight implementation of the Minecraft server where exact vanilla functionality is not needed or higher performance is desired than the official software can deliver. Glowstone makes use of a thread-per-world model and performs synchronization only when necessitated by APIs.

Glowstone++ is an experimental work in progress, and is not ready for production servers.
